## Что строим

Браузерная игра-симулятор марсианской теплицы. Игрок управляет климатом (температура, влажность, кислород), сажает инопланетные растения, реагирует на катаклизмы и собирает урожай за космо-кредиты. Прогресс и рекорды хранятся локально (localStorage).

## Дизайн

Мультяшный sci-fi: тёмный космический фон с лёгким градиентом и звёздами, неоновые акценты (циан, маджента, лайм), скруглённые углы, мягкие свечения (box-shadow с цветом акцента). Шрифт-заголовок — `Space Grotesk`, тело — `Inter`. Все цвета — через семантические токены в `src/styles.css` (`--neon-cyan`, `--neon-magenta`, `--neon-lime`, градиенты, свечения).

```text
│   ┌──┬──┬──┐                                 │
│   │🌱│  │🍄│   Грядка 3x3                    │
│   ├──┼──┼──┤   (клик → посадка/сбор)         │
│   │  │🔥│  │                                 │
│   ├──┼──┼──┤                                 │
│   │  │  │  │                                 │
│   └──┴──┴──┘                                 │
│                                              │
│   🌡 Температура  ▬▬▬●▬▬▬  +20°C             │
│   💧 Влажность    ▬▬●▬▬▬▬  40%               │
│   🫧 Кислород     ▬▬▬▬●▬▬  60%               │
│                                              │
│   [🛒 Магазин семян]   [📜 События]          │
└──────────────────────────────────────────────┘
```

- **Грядка**: 3x3 CSS Grid. Каждая ячейка показывает растение (эмодзи/SVG), мини прогресс-бар роста, смайл настроения (🟢/🟡/🔴). Пустая ячейка — `+` для посадки.
- **Ползунки**: три `input[type=range]`. Цвет трека меняется через CSS-переменные в зависимости от значения (температура: синий→зелёный→красный).
- **Магазин** (Dialog/Sheet shadcn): сетка карточек семян с ценой, идеальными параметрами и временем роста. Кнопка «Купить» добавляет в инвентарь.
- **Лидерборд** (Dialog): топ-10 рекордов из localStorage с именем и очками.
- **Тосты** (sonner) для событий: «🌪 Пылевая буря! Температура падает», «✨ Урожай созрел».

## Игровая логика

**6 растений** с разными требованиями:
1. Лавовый куст 🔥 — +40°C, 10% влажности, 50% O₂, 60с, цена 50, очки 10
2. Неоновый гриб 🍄 — 0°C, 90% влажности, 30% O₂, 45с, цена 40, очки 8
3. Космо-кактус 🌵 — +25°C, 20% влажности, 40% O₂, 30с, цена 20, очки 5
4. Ледяная роза 🌸 — -20°C, 70% влажности, 60% O₂, 90с, цена 80, очки 15
5. Звёздный лишайник ✨ — +10°C, 50% влажности, 80% O₂, 60с, цена 60, очки 12
6. Плазма-лиана 🌿 — +35°C, 60% влажности, 70% O₂, 120с, цена 100, очки 20

**Стадии роста**: Семя → Росток → Взрослое → Урожай (4 стадии). Длительность стадии = `totalTime/3`. Скорость роста = `1.0` если все 3 параметра в пределах ±10 от идеала, `0.5` если ±20, `0` и риск болезни если дальше.

**Настроение**: средняя «дистанция» текущих параметров от идеала → зелёный/жёлтый/красный.

**Качество урожая** при сборе: среднее настроение за всё время роста → 1/2/3 звезды → множитель кредитов и очков (×1, ×1.5, ×2).

**Случайные события**: каждые 30–60с триггерится одно из:
- Пылевая буря: −15°C на 10с
- Солнечная вспышка: +20°C на 10с
- Утечка кислорода: −20% O₂ на 10с
- Метеоритный дождь: −10% влажности на 10с

Эффект применяется как оффсет к показаниям климата (игрок видит, что значения «уехали», и подкручивает ползунки).

**Тик игры**: один `setInterval` на 1с, обновляет всё растения, проверяет события, сохраняет состояние.

## Сохранение

Один Zustand-стор с middleware `persist` в localStorage (`mars-greenhouse-v1`):
- credits, totalScore, playerName
- garden: массив из 9 ячеек (растение, посажено когда, накопленный «happiness score»)
- climate: {temp, humidity, oxygen}
- inventory: купленные семена
- activeEvent: текущее событие и его таймер
- leaderboard: топ-10 `{name, score, date}[]`

При первом заходе — диалог «Введи имя космо-агронома», 250 стартовых кредитов.
