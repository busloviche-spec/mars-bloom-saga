# 2D-версия: марсианская теплица в разрезе

Полностью заменяю карточный `Garden` на анимированную 2D-сцену "вид сбоку". Логика игры (store, tick, события, вредители, урожай, апгрейды) остаётся без изменений — меняется только слой представления.

## 1. Новая сцена `GreenhouseScene.tsx`

Файл: `src/components/game/GreenhouseScene.tsx`

- Контейнер на всю ширину `max-w-5xl`, высота ~420px (адаптивно), `relative overflow-hidden`, `rounded-2xl`.
- Слои (снизу вверх, абсолютное позиционирование):
  1. **Небо Марса** — градиент от тёмно-бордового к оранжево-ржавому + звёзды (существующий `.stars`).
  2. **Дальние скалы/кратеры** — CSS-силуэты (SVG `path` или `clip-path polygon`), медные тона.
  3. **Поверхность Марса** — песчаная полоса внизу с рябью (повторяющийся градиент).
  4. **Ряд изо-боксов** — flex-row по низу сцены, каждый бокс = стеклянный "аквариум".
  5. **Слой червя** — абсолютный SVG/div поверх песка, ползёт от края к боксу-жертве.
  6. **Слой погоды** — при активном событии: пыльная буря (drift частиц), метеоры и т.п.

## 2. Компонент `GreenhouseBox2D.tsx`

Один бокс в разрезе:

```text
 ┌──────────────┐   ← крышка (медная рамка + светодиод уровня)
 │  🌱 растение │   ← стеклянные стенки (border + backdrop-blur)
 │  ▓▓▓▓▓░░░░░  │   ← прогресс-полоса роста внутри
 └──────────────┘
   [🌡][💧][🫧]   ← мини-ползунки под боксом
```

- Ширина ~150–180px, высота ~180px.
- Растение — крупный emoji со `stageEmoji`, `scale` зависит от `cell.progress`, покачивание (`animate-[sway]`).
- Готовый урожай — свечение (`shadow-[0_0_24px_var(--neon-lime)]`), пульсация, клик = сбор.
- Пустой бокс — иконка "+" по центру, клик открывает `PlantSeedDialog`.
- Настроение (emoji + цвет) — маленький бейдж в углу крышки.
- Кнопка апгрейда бокса ⚡ + бейдж уровня — на крышке.
- Компактные ползунки климата 🌡/💧/🫧 отрисовываются под боксом (переиспользую `BoxClimateControls` в компактном режиме — добавлю prop `compact` для укороченной раскладки).
- Червь: если `pest.boxId === box.id`, рендерится анимированный червь, ползущий из-под песка к боксу и грызущий его; клик по червю = `squashPest`.

## 3. Червь в 2D

- SVG длиной ~80px, сегменты кругами разного диаметра, "змейка" body-wave через CSS-переменную `--t`.
- Позиция: `bottom: 0`, `left`: интерполируется от края сцены к центру целевого бокса за первые ~30% biteProgress, затем закрепляется у бокса и "грызёт" (shake).
- Клик — событие `stopPropagation`, вызывает `squashPest()` и sfx.

## 4. Кнопка "Новый изо-бокс"

Отдельный "пустой слот" в конце ряда — светящаяся пунктирная рамка формы бокса с "+" и ценой. При достижении `MAX_BOX_COUNT` слот скрывается.

## 5. Скролл при большом числе боксов

`overflow-x-auto` на ряду боксов, snap-x. На мобильных — свайп; на десктопе — все помещаются.

## 6. Изменения в существующих файлах

- `src/components/game/GreenhouseGame.tsx` — заменить `<Garden />` на `<GreenhouseScene onPlant=... />`. Всё остальное (TopBar, инструкция, кнопки Магазин/Сундуки, диалоги) без изменений.
- `src/components/game/ClimateControls.tsx` — добавить опциональный prop `compact?: boolean`; в compact режиме прячем подписи, делаем ползунки одной строкой с иконкой и малым значением. Tooltip остаётся.
- `src/components/game/Garden.tsx` — оставляю файл, но он больше не используется (или удаляю, чтобы не плодить мёртвый код — удалю).
- `src/styles.css` — добавить keyframes:
  - `@keyframes sway` — покачивание растения.
  - `@keyframes worm-crawl` — body-wave червя.
  - `@keyframes dust` — марсианская пыль.
  - Утилита `.mars-ground` (повторяющийся песочный градиент).

## 7. Что НЕ трогаю

- `src/game/store.ts`, `useGameTick.ts`, `events.ts`, `plants.ts` — вся игровая логика без изменений.
- Диалоги посадки, магазина, сундуков, лидерборда, инструкции.
- Марсианская цветовая тема из прошлых шагов.

## Технические детали

- Позиционирование через flex + absolute layers; никакого canvas/WebGL — чистый JSX + CSS.
- Все цвета через семантические токены (`var(--neon-magenta)` = марсианский красный, `var(--neon-cyan)` = медный, `var(--neon-lime)` = песочный).
- Анимации — CSS keyframes, без внешних библиотек.
- Клики на боксе используют event delegation: клик по стенкам → посадка/сбор, клик по червю → раздавить (stopPropagation).
