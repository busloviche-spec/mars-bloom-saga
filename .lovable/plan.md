## 3. Червь-вредитель

Игровая логика (`src/game/store.ts` + новый `src/game/pest.ts` при необходимости):
- Новое поле состояния: `pest: { boxId: string; spawnedAt: number; biteProgress: number } | null` и таймер `lastPestCheck`.
- В `tick()` (или отдельном интервале в `useGameTick`): если `pest === null`, есть хотя бы один бокс с растущим растением и прошло 25–60с с прошлой попытки — с шансом ~15% выбрать случайный такой бокс и заспаунить червя.
- Пока червь сидит на боксе: каждый тик уменьшает `progress` растения (например, −0.02/сек) и увеличивает `biteProgress`. Если `biteProgress ≥ 1` без клика — растение уничтожается (сброс ячейки), тост «Червь съел урожай».
- Действие `squashPest()`: вызывается по клику; выдаёт бонус (например, +25💰 и +10 очков, с шансом 10% — «сундук»), обнуляет червя, играет звук.

UI:
- В `PlantCell`/`GreenhouseBoxCard`: если `pest.boxId === box.id`, поверх ячейки — абсолютно позиционированный кликабельный эмодзи 🪱 с анимацией `animate-bounce`/дрожание и обводкой; клик → `squashPest()`.
- Тост при появлении червя и при уничтожении (аналогично `greenhouse:harvest`).

Звук (`src/game/sounds.ts`):
- Новый `sfx.pest()` — короткий шумовой «грызущий» звук: `noise(0.12, 0.15, 800)` серией из 3–4 повторов с интервалом 0.15с; проигрывается при спавне и повторно каждые ~2с, пока червь жив (через `setInterval` в `useGameTick` или в компоненте с червём).
- `sfx.squash()` — резкий короткий «плюх» (низкочастотный noise + tone 200→60Hz).

## 4. Разблокировка семян за звёзды достижений

Модель звёзд:
- Ввести `starsEarned: number` в сторе — суммарные звёзды со всех сборов (инкрементим в `harvest` на значение `stars` 1/2/3).
- Персистить наравне с `totalScore`.

Каталог растений:
- В `src/game/plants.ts` расширить тип `Plant` полем `unlockStars?: number` (для обычных семян; AI-семена из сундуков не блокируются).
- Существующие 6 базовых растений получают пороги: например, 0, 0, 5, 15, 40, 100 (точные числа подберём по балансировке; первые 2 доступны сразу).

Магазин (`SeedShopDialog.tsx`):
- Для каждого растения проверяем `starsEarned >= unlockStars`.
- Разблокированные — как сейчас (кнопки «Купить»/«⚡ Апгрейд»).
- Заблокированные — карточка серая, вместо кнопки покупки бейдж `🔒 Нужно ⭐ N (у вас M)` и полоска прогресса `min(M/N, 1)`; апгрейд тоже скрыт.
- Сортировка: сначала открытые, затем заблокированные по возрастанию `unlockStars`.

TopBar/достижения:
- В `TopBar.tsx` добавить компактный индикатор `⭐ {starsEarned}` рядом с монетами/очками, чтобы игрок видел прогресс к разблокировке.
