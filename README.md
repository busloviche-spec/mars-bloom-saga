# Марсианская теплица

Игра-симулятор для детей " Марсианская теплица"- выращивание растений на Марсе с возможностью ИИ-генерации редких семян растений

## Как играть
1. Открой ссылку https://github.com/busloviche-spec/mars-bloom-saga
2. [Прочитай правила игры, внеси в строку ввода имя игрока,зайди в магазин семян, купи семена и начинай выращивать их]
3. [Получишь монеты и звезды за сбор урожая. За монеты можешь приобрести новые семена растений, прокачать их, дополнительно купить и прокачать боксы   ]
